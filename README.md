# EduAgent-IQ: Smart Study Companion
EduAgent-IQ is an AI-powered assistant designed to enhance personalized learning and conceptual clarity.
Key Features:
Adaptive Learning: Explanations tailored for School, Intermediate, and College levels.
Smart Quiz Engine: Generates custom quizzes and provides instant evaluation/scoring.
Concept Comparison: Side-by-side comparison of complex topics for better understanding.
Optimized Accuracy: Enhanced prompt engineering to ensure verified, fact-focused responses.
Goal: Providing students with a powerful reasoning agent for effective exam preparation and conceptual mastery.
